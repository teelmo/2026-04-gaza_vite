# Pattern: d3 Data Visualization

## Visualisointitavan valinta

| Visualisointityyppi | Toteutus | Miksi |
|---------------------|----------|-------|
| Palkkikaaviot (bar charts) | HTML `<div>` -elementit | Kevyt, nopea, helppo tyylitellä Tailwindilla |
| Viivakaaviot, alueet | D3 + React | Tarvitaan path-laskentaa |
| Kartat, kompleksiset muodot | D3 + React | Tarvitaan geo-projektioita |
| Piirakat, donitit | D3 + React | Tarvitaan arc-generaattoreita |

## Palkkikaaviot div-elementeillä

```tsx
// BarChart.tsx
interface DataPoint {
  label: string;
  value: number;
}

interface Props {
  data: DataPoint[];
  maxValue: number;
}

const BarChart = ({ data, maxValue }: Props) => {
  return (
    <div className="flex flex-col gap-2">
      {data.map((item) => (
        <div key={item.label} className="flex items-center gap-2">
          <span className="w-[100px] shrink-0">{item.label}</span>
          <div
            className="h-6 rounded-sm bg-[var(--yds-infographic-news-default-2)] transition-[width] duration-300"
            style={{ width: `${(item.value / maxValue) * 100}%` }}
          />
          <span className="text-[var(--yds-color-text-default)]">{item.value}</span>
        </div>
      ))}
    </div>
  );
};
```

## D3 + React: Työnjako

**Perusperiaate:** D3 laskee, React renderöi.

| D3:n rooli | Reactin rooli |
|------------|---------------|
| Skaalojen laskenta | JSX-elementtien renderöinti |
| Path-generaattorit | `useMemo` – reaktiivinen päivitys |
| Geo-projektiot | DOM-hallinta |

### Esimerkki: Viivakaavio

```tsx
import { useMemo } from "react";
import * as d3 from "d3";

const LineChart = ({ data, width, height }: Props) => {
  const margin = { top: 20, right: 20, bottom: 30, left: 40 };

  const linePath = useMemo(() => {
    const xScale = d3.scaleTime()
      .domain(d3.extent(data, (d) => d.date) as [Date, Date])
      .range([margin.left, width - margin.right]);

    const yScale = d3.scaleLinear()
      .domain([0, d3.max(data, (d) => d.value) || 0])
      .range([height - margin.bottom, margin.top]);

    const lineGenerator = d3.line<DataPoint>()
      .x((d) => xScale(d.date))
      .y((d) => yScale(d.value));

    return lineGenerator(data) || "";
  }, [data, width, height]);

  return (
    <svg width={width} height={height}>
      <path d={linePath} fill="none" stroke="var(--yds-infographic-news-default-2)" strokeWidth={2} />
    </svg>
  );
};
```
